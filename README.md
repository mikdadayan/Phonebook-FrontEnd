# phonebook-backend-api

## To run this application

1. `git clone https://github.com/mikdadayan/tournaments-search-for-deploy-backend.git`
2. `npm install`
3. `npm start`

## Or Simply run

1. `git clone https://github.com/mikdadayan/tournaments-search-for-deploy-backend.git`
2. `docker-compose up --build`
